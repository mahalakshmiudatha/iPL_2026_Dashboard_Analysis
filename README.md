# 🏏 IPL 2026 Dashboard Analysis using Power BI
# Overview
This project is an interactive Power BI dashboard created using IPL 2026 match data. As a cricket enthusiast, I wanted to combine my passion for cricket with data analytics and explore real-world sports data through interactive visualizations.

The dashboard provides insights into team performance, venue statistics, toss decisions, player achievements, and season standings. It demonstrates how data can be transformed into meaningful insights using Power BI.

## Project Objective

The objective of this project is to analyze IPL 2026 data and create an interactive dashboard that helps users understand team performance, player statistics, venue trends, and overall tournament outcomes.

## Dashboard Features

- Total Matches Analysis
- Total Teams Analysis
- Total Venues Analysis
- Champion Team Identification
- Team Wise Wins Analysis
- Toss Decision Analysis
- Venue Wise Match Distribution
- Top 10 Man of the Match Players
- IPL 2026 Points Table
- Interactive Team and Venue Filters

## Tools Used

- Power BI
- Power Query
- DAX
- Excel / CSV Dataset

## Skills Demonstrated

- Data Cleaning
- Data Transformation
- Data Modeling
- DAX Calculations
- KPI Creation
- Dashboard Design
- Interactive Reporting
- Data Visualization

## Key Insights

- Identified the IPL 2026 Champion Team
- Compared team-wise wins across the tournament
- Analyzed toss decision patterns
- Explored venue-wise match distribution
- Identified top-performing players based on Man of the Match awards
- Reviewed season standings through the points table

Cricket is one of my favorite sports, and this project allowed me to apply my data analytics skills to a domain I enjoy. Working with IPL data helped me gain practical experience in creating professional dashboards and extracting meaningful insights from real-world datasets.
